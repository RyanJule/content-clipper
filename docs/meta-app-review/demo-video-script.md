# Demo Video Script for Meta App Review

This script guides you through recording a demo video demonstrating Content Clipper's Instagram integration for Meta app review.

## Video Requirements

- **Duration:** 3-5 minutes
- **Format:** MP4, MOV, or WebM
- **Resolution:** Minimum 720p (1080p recommended)
- **Audio:** Clear narration or text overlays explaining actions
- **Content:** Screen recording showing all permission use cases

## Recommended Tools

- **Screen Recording:**
  - macOS: QuickTime Player (built-in)
  - Windows: OBS Studio, Camtasia, or built-in Xbox Game Bar
  - Chrome Extension: Loom, Screencastify
- **Video Editing:** iMovie, Adobe Premiere, DaVinci Resolve, or Camtasia
- **Microphone:** Built-in or external USB microphone

---

## Script Overview

The video should demonstrate:

1. Login and authentication (15 seconds)
2. Account connection with OAuth (45 seconds)
3. Content upload and publishing (90 seconds)
4. Comment management (30 seconds)
5. Message management (30 seconds)
6. Analytics viewing (30 seconds)
7. Conclusion (15 seconds)

**Total Time:** ~4 minutes

## Recording Tips

### Before Recording

1. **Prepare everything:**
   - Test account logged in
   - Test video ready to upload
   - Test comment added to a post
   - Test DM sent to account
   - Clear browser cookies/cache for clean demo

2. **Clean up:**
   - Close unnecessary browser tabs
   - Hide bookmarks bar
   - Use incognito/private window if possible
   - Disable notifications

3. **Set up screen:**
   - Zoom level: 100% or 110% (not too large)
   - Window size: Full screen or large enough to see clearly
   - Hide personal information

### During Recording

1. **Move slowly:**
   - Don't rush through actions
   - Pause briefly before clicking
   - Let pages load fully

2. **Speak clearly:**
   - Articulate each word
   - Maintain consistent volume
   - Pause between sections

3. **Highlight actions:**
   - Circle cursor around important items
   - Use on-screen annotations if possible
   - Point out key elements

### After Recording

1. **Edit video:**
   - Trim any mistakes or dead time
   - Add text overlays for key points
   - Add arrows or highlights to emphasize features
   - Include intro/outro cards

2. **Add captions:**
   - Include closed captions for accessibility
   - Highlight permission names when demonstrating them

3. **Review:**
   - Watch entire video
   - Check audio levels
   - Verify all permissions are demonstrated
   - Ensure video quality is clear

## Upload and Submit

Once your video is ready:

1. **Upload to:**
   - YouTube (unlisted or private)
   - Vimeo
   - Direct file upload to Meta (if supported)

2. **Include in app review:**
   - Provide video URL in app review form
   - Add to "App Review Notes" section
   - Reference video in permission justifications

3. **Keep accessible:**
   - Don't delete video until review is complete
   - Ensure privacy settings allow Meta reviewers to view
   - Test link in incognito mode to verify access
